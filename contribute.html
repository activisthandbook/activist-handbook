Activist Handbook is written by activists like you. We are building this knowledge base together. You can start contributing too by creating an account!

  • Write an article: Any contribution is valued, no matter how small. Create a completely new page, add some resources to a page, or correct a typo.
  • Translate a page
  • Moderate: Every edit is saved forever. This makes it easier for us to moderate. You can view version history of pages on our website, as well as via our backup at GitHub.
  • Become a developer: Activist Handbook is an open source project. We very much welcome you to contribute. We made a page for developers where we document how our platform was built.

If you would like to become more involved, you can join our volunteer team! Anyone who is interested is welcome to become a volunteer. Learn more about how to join us.

How to use the website

We want anyone to be able to contribute. That is why we made our platform super easy to use. Check out the images below to learn how to use the website. If you have any questions, do not hesitate to reach out! We are happy to help: contact@activisthandbook.org
